## Agent skills

### Issue tracker

Issues 存在 GitHub Issues 中（仓库 `chinwe/trunk`），通过 `gh` CLI 操作。外部 PR 也作为 triage 需求入口。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用默认标签名：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### Domain docs

单一上下文：`CONTEXT.md` + `docs/adr/` 位于仓库根目录。详见 `docs/agents/domain.md`。
